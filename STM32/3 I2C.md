# I2C
## 第14课 [I2C]基础知识
1. I2C是一种总线结构，1个STM32可以与大量外部I2C设备连接。
2. I2C有两条通信线：SCL（Serial Clock）时钟线和SDA（Serial Data）数据线，所有设备的SCL接到一下，SDA接到一起。记得SCL和SDA都要接上拉电阻。
   ![alt text](<assets/3 I2C/image-7.png>)
3. SCL和SDA都使用开漏输出模式，是为了实现与运算。
   ![alt text](<assets/3 I2C/image.png>)
4. I2C通信过程：起始位、寻址、数据传输、停止位。
    1. 起始位。在SCL高电平时，主站向SDA发送一个低电平。
    ![alt text](<assets/3 I2C/image-1.png>)
    2. 寻址。主站向SDA发送8位数据：从站的地址7位+1位R/W#位。（从站地址分为7位和10位，我们这里用7位。）R/W#=0代表主站向从站写数据，R/W#=1代表主站向从站读数据（也就是从站向主站写数据）。发送完之后将SDA电平拉高。
    3. 寻址应答：从站向SDA发送1位应答信号ACK（acknowledge），将SDA电平拉低。
    ![alt text](<assets/3 I2C/image-2.png>)
    4. 主站写数据传输：主站向SDA发送一个字节，然后将SDA拉高。
    5. 主站写数据传输应答：从站发送ACK，将SDA拉低。
    6. 主站读数据传输：从站向SDA发送一个字节，然后将SDA拉高。
    7. 主站读数据传输应答：主站发送ACK，将SDA拉低。
    ![alt text](<assets/3 I2C/image-3.png>)
    8. 停止位：在SCL高电平时，主站向SDA发送一个高电平。
    ![alt text](<assets/3 I2C/image-4.png>)
    9.  实例如下：
    ![alt text](<assets/3 I2C/image-5.png>)
5. I2C通信的波特率：分成SM和FM等5种模式。占空比分为2：1和16：9，一般选2：1。
   ![alt text](<assets/3 I2C/image-6.png>)
## 第15课 [I2C]简单数据收发实验
### 一、HAL库函数
1. I2C发送函数。
   ```c
   HAL_StatusTypeDef HAL_I2C_Master_Transmit(I2C_HandleTypeDef *hi2c, 
                                         uint16_t DevAddress, 
                                         uint8_t *pData, 
                                         uint16_t Size, 
                                         uint32_t Timeout)
   //参数hi2c：i2c接口的句柄指针
   //DevAddress:从站地址
   //*pData：要发送的数据
   //Size：发送的字节数
   //Timeout：超时时间，单位是ms
   // 函数返回值为：HAL_OK 发送成功；HAL_ERROR 发送出错；HAL_BUSY 接口忙。
   ```
   举例
   ```c
   uint8_t commands[] = {0x00,0x8d,0x14,0xaf,0xa5};
   HAL_I2C_Master_Transmit(I2C_HandleTypeDef &hi2c1, //i2c句柄
                                         0x78, //从站地址
                                         commands, //发送的数据
                                         sizeof(commands), //发送的字节数
                                         HAL_MAX_DELAY)//超时时间无限
   ```
2. I2C接收函数。
   ```c
   HAL_StatusTypeDef HAL_I2C_Master_Receive(I2C_HandleTypeDef *hi2c, 
                                       uint16_t DevAddress, 
                                       uint8_t *pData, 
                                       uint16_t Size, 
                                       uint32_t Timeout);
   //参数含义和返回值与上面的发送函数相同
   ```
   距离
   ```c
   uint8_t dataRcvd;
   HAL_I2C_Master_Receive(&hi2c1,
                        0x78,
                        &dataRcvd,
                        1,
                        HAL_MAX_DELAY);
   ```
